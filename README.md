# Hospital Management System (DSA Project)

##  Overview

This project is a command-line based Hospital Management System developed in C++.
It demonstrates the practical implementation of Data Structures and Algorithms (DSA).

## Objectives

* Manage doctors and nurses records
* Handle patient appointments
* Implement queue system for patients
* Maintain history using stack
* File handling for data storage

##  Data Structures Used
* **Arrays** → Store doctors and nurses
* **Queue** → Manage patient waiting line (FIFO)
* **Stack** → Store patient history (LIFO)
* **File Handling** → Save and retrieve data

## ⚙️ Features

* Add / View Doctors
* Add / View Nurses
* Patient Appointment System
* Emergency Handling
* Patient History Tracking

##  Project Structure

* `main.cpp` → Main program logic
* `.txt files` → Data storage
